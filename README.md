# FixIt — Demo
Demo mobil-friendly app (React + Vite) pentru un asistent digital de reparații casnice și conectare cu muncitori.

Quick start:

1. Install dependencies

```bash
cd /home/alex/Documente/Ifixit
npm install
```

2. Run dev server

```bash
npm run dev
```

Notes:
- Acesta este un demo. Nu sunt procesate plăți.
- Datele muncitorilor sunt demo și salvate în `localStorage`.
- AI-ul este un mock local; poate fi extins cu API-uri externe.
